Neste projeto, realizado na Universidade de Sorocaba no segundo semestre de 2023, concebemos uma empresa fictícia denominada Trail Life, uma transportadora privada especializada em entregas para o setor da saúde, abrangendo farmácias, clínicas e hospitais. Desenvolvemos um sistema abrangente de gerenciamento de transporte, dividido em quatro ambientes distintos.

Landing Page

Contruída para a captação de clientes

Uma página web que serve para otimizar o marketing da transportadora. Sendo uma abordagem abrangente e estratégica, capturando a atenção de novos clientes e colaboradores. Nela, possíveis novos clientes podem entrar em contato com a transportadora.

AWS SES

A Landing Page Trail Life utiliza o Amazon Simple Email Service (SES) para o envio de e-mails, proporcionando confiabilidade aos provedores de e-mail.

O Amazon SES é conhecido por sua confiabilidade devido à sua infraestrutura sólida e práticas recomendadas. Uma escolha segura para entregas consistentes.

System Page

Desenvolvida para otimizar a gestão e trazer um gerenciamento excepcional

Contendo 3 tipos de acesso do usuário, sendo elas, todas (o usuário tem acesso total ao sistema), ver (o usuário só pode ver/ler as informações) e escrever (o usuário tem a permissão de controlar algumas ações no sistema) que é a permissão mais comum dos administradores. Dentro desse ambiente temos o controle de cargas (adicionar, editar, ver e excluir), o controle de motoristas (adicionar, editar, ver e excluir), o controle de clientes (adicionar, editar, ver e excluir) e também o controle de administradores(adicionar, editar, ver e excluir), no qual é apenas liberado para quem tem o tipo de acesso 'todas'.

AWS EC2

O Trail Life System utiliza instâncias Amazon EC2 como parte fundamental de sua infraestrutura tecnológica, garantindo desempenho e flexibilidade na execução de suas operações.

O Amazon EC2, ou Elastic Compute Cloud, é um serviço de computação em nuvem da Amazon Web Services. Ele oferece capacidade computacional sob demanda, permitindo aos usuários criar e executar instâncias virtuais de servidores de acordo com suas necessidades específicas. Essa flexibilidade proporciona agilidade e eficiência na gestão de recursos de computação na nuvem.

Desenvolvimento Front-End

Técnicas avançadas

Na criação do front-end foi utilizado HTML, CSS, JavaScript, Bootstrap e Sass, centrando-se especialmente no desenvolvimento do front-end. Técnicas avançadas de desenvolvimento web foram cuidadosamente empregadas, utilizando HTML para a estruturação do conteúdo, CSS para o design e estilo visual, e JavaScript para a interatividade e dinamismo da interface. A integração do Bootstrap proporcionou uma abordagem ágil e responsiva no desenvolvimento do layout, enquanto o Sass contribuiu para uma gestão mais eficiente e modular do código CSS. Essas escolhas tecnológicas contemporâneas resultaram em um front-end robusto, esteticamente aprimorado e alinhado às melhores práticas de desenvolvimento web.

Desenvolvimento Back-End

Soluções tecnológicas

O back-end foi estruturado de forma a fornecer uma base sólida para a integração eficiente com o front-end. A utilização do PHP permitiu a implementação de lógicas complexas e manipulação de dados, contribuindo para a construção de um sistema coeso e funcional. Essa escolha tecnológica específica proporcionou não apenas a capacidade de processar dados de maneira eficaz, mas também a flexibilidade necessária para atender aos requisitos dinâmicos do projeto.

Banco de Dados

Sistema de banco de dados altamente protegido

O MySQL foi estruturado de maneira a oferecer uma base sólida e confiável para a persistência de informações, assegurando a criptografia gerada pela aplicação, permitindo consultas eficientes e contribuindo para a integridade e segurança dos dados. A modelagem do banco de dados foi desenvolvida considerando as necessidades específicas do projeto, proporcionando uma arquitetura que suporta de maneira eficiente as operações do back-end em PHP.
